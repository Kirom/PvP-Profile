# PvP Profile v1.0.8 Release Notes

**Release Date:** December 17, 2025  
**Version:** 1.0.8

## Overview

This update ensures compatibility with the latest World of Warcraft MoP Classic patch.

## Changes

### Compatibility Update

**Interface Version Update**
- **What Changed**: Updated the Classic addon's interface version from 50502 to 50503
- **Why**: This ensures compatibility with the latest World of Warcraft MoP Classic patch (5.5.3)
- **Technical Details**:
  - Updated `PvPProfile_Classic.toc` to use interface version 50503
  - No changes to Retail version

## Known Issues

None reported for this version.

## Support

If you encounter any issues with this release, please:
1. Check the [GitHub Issues](https://github.com/Kirom/PvP-Profile/issues) page
2. Ensure you're using the correct version for your WoW client (retail vs classic)
3. Try disabling other addons to check for conflicts
4. Check the [Discord](https://discord.gg/5sMTavfNsE)
