# PvP Profile v1.0.10 Release Notes

**Release Date:** February 15, 2026  
**Version:** 1.0.10

## Overview

This update provides compatibility with the latest World of Warcraft Midnight patch by updating the interface version for retail.

## Changes

### Compatibility Update

**Interface Version Update**
- **What Changed**: Updated the Retail addon's interface version from 120000 to 120001
- **Why**: This ensures compatibility with the latest World of Warcraft Midnight patch (12.0.1)
- **Technical Details**:
  - Updated `PvPProfile.toc` to use interface version 120001
  - No changes to Classic version

## Known Issues

None reported for this version.

## Support

If you encounter any issues with this release, please:
1. Check the [GitHub Issues](https://github.com/Kirom/PvP-Profile/issues) page
2. Ensure you're using the correct version for your WoW client (retail vs classic)
3. Try disabling other addons to check for conflicts
4. Check the [Discord](https://discord.gg/5sMTavfNsE)
