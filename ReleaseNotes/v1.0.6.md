# PvP Profile v1.0.6 Release Notes

**Release Date:** November 10, 2025  
**Version:** 1.0.6

## Overview

This update provides compatibility with the latest World of Warcraft MOP patch by updating the interface version for classic.

## Changes

### Compatibility Update

**Interface Version Update**
- **What Changed**: Updated the addon's interface version from 50501 to 50502
- **Why**: This ensures compatibility with the latest World of Warcraft MOP patch
- **Technical Details**:
  - Updated `PvPProfile_Classic.toc` to use interface version 50502
  - Maintains full functionality across all supported WoW versions
  - No breaking changes to existing features

## Known Issues

None reported for this version.

## Support

If you encounter any issues with this release, please:
1. Check the [GitHub Issues](https://github.com/Kirom/PvP-Profile/issues) page
2. Ensure you're using the correct version for your WoW client (retail vs classic)
3. Try disabling other addons to check for conflicts
4. Check the [Discord](https://discord.gg/5sMTavfNsE)
